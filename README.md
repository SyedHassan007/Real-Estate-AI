# 🏙️ Dubai Real Estate AI & Analytics Project

## 🚀 Overview
This project is an end-to-end AI/ML and data analytics solution designed to analyze real estate data and predict property prices in Dubai.

## 🧠 Features
- Synthetic real estate dataset generation
- Data cleaning and feature engineering
- Machine learning models (Linear Regression, Random Forest)
- Model evaluation (MAE, RMSE, R²)
- SQLite database integration
- Power BI-ready datasets
- Flask API for predictions
- Streamlit web app for interactive UI

## 🛠️ Tech Stack
- Python (Pandas, NumPy, Scikit-learn)
- SQL (SQLite)
- Power BI
- Excel
- Flask
- Streamlit

## 📊 Machine Learning
- Regression models for price prediction
- Feature engineering (Rental Yield, Accessibility Score)
- Model comparison and selection
